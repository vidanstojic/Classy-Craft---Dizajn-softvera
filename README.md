# Classy Craft: UML & Mind Mapping Tool

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Maven](https://img.shields.io/badge/Apache_Maven-C71A36?style=for-the-badge&logo=apache-maven&logoColor=white)

**Classy Craft** is a Java-based desktop application developed as part of the **"Software Design"** university course. The project focuses on implementing robust software architecture through the **MVC (Model-View-Controller)** pattern and various advanced design patterns.

The application serves a dual purpose: creating structured **UML diagrams** for software engineering and flexible **Mind Maps** to assist with studying and brainstorming.

---

## Design Patterns Applied

A core requirement of this project was the practical application of industry-standard design patterns:

* **Singleton**: Manages global application states and configurations.
* **Observer**: Facilitates real-time UI updates when the underlying model changes.
* **State Pattern**: Manages different interaction states (e.g., selection, drawing, and moving elements).
* **Command Pattern**: Implements an extensible system for actions, enabling Undo/Redo functionality.
* **Composite Pattern**: Handles hierarchical structures within maps and diagrams.
* **Factory Method**: Dynamically instantiates various diagram elements like classes and relationships.

---

## Key Features

### UML Modeling
* **Class Creation**: Design interfaces, abstract classes, and standard classes.
* **Relationship Mapping**: Support for Association, Generalization, and Dependency relations.
* **Advanced Editing**: Move topics, edit text, and center the view on specific elements.
* **Compatibility**: Import existing UML files or export your work for future use.
* **Project Organization**: Group multiple UMLs into unified projects.
* **Custom Templates**: While no default templates are enforced, users can create and save their own styling patterns.
* **Flexible Export**: Save your UML diagrams as high-quality images or raw UML files.

## Getting Started

### Prerequisites
* Java JDK (8 or higher recommended)
* Apache Maven

### Installation & Execution
1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/vidanstojic/Classy-Craft---Dizajn-softvera.git](https://github.com/vidanstojic/Classy-Craft---Dizajn-softvera.git)
    ```
2.  **Open the project**: Import the folder into your preferred IDE (IntelliJ IDEA or VS Code).
3.  **Build the project**: Let Maven resolve all dependencies.
4.  **Run the application**: Locate and execute the main entry point:
    `src/main/java/.../AppCore.java`
